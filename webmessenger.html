The Institut national de l'audiovisuel (or INA, French for National Audiovisual Institute), is a repository of all French radio and television audiovisual archives. Frequently asked questions
How do I access MSN Web Messenger from another computer?
In one of the web browsers listed above, enter http://webmessenger.msn.com/?mkt=en-us in the address bar. In Microsoft Internet Explorer, you can add this site to your favorites by pressing CTRL+D.
What is the difference between Web Messenger and Messenger?
MSN Messenger is a fully featured instant messaging program that you install on your own computer or one you have permission to install on. MSN Web Messenger enables you to quickly and easily use basic instant messaging features on a web browser on any computer without installing any software.
When will the beta-testing period end?
The full version of MSN Web Messenger will be released later this year. It will incorporate suggestions and feedback from our users. If you have feature requests or suggestions for improving MSN Web Messenger, please click the Provide Feedback link in the "I want to..." section of the MSN Web Messenger page and tell us what you think!
Why do I sometimes see the error message "We can't sign any more people in right now. Please try signing in again later"?
Because this is a trial, we are limiting the number of simultaneous connections to MSN Web Messenger. We encourage you to try again later when the service is less busy.
